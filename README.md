# AI Content Marketing Using UGC Ads

## Project Overview

This project demonstrates how AI tools can be used to create User Generated Content (UGC) advertisements for a real product. The project focuses on generating engaging marketing content, platform-specific advertisements, and conversion-focused copy using prompt engineering techniques.

## Product Selected

boAt Airdopes Wireless Earbuds

## Objective

To create a complete AI-generated UGC Ad Content Pack including:
- Ad Hooks
- UGC Ad Scripts
- Instagram Reel Content
- Facebook Ad Copy
- YouTube Shorts Script
- Call-To-Actions (CTAs)
- Brand Voice Adaptations

## Tools Used

- Claude AI
- ChatGPT
- Google Docs / MS Word
- GitHub

## Skills Gained

- AI-Driven Content Marketing
- Prompt Engineering
- UGC Advertising
- Conversion Copywriting
- Social Media Marketing
- Brand Voice Adaptation

## Learning Outcomes

- Learned AI-driven content creation techniques.
- Practiced prompt engineering for marketing use cases.
- Generated platform-specific advertising content.
- Explored conversion-focused copywriting strategies.
- Adapted content for different brand voices and audiences.

## Conclusion

This project demonstrates how AI can assist in creating effective UGC advertisements and marketing campaigns. By leveraging prompt engineering and generative AI tools, engaging and audience-focused content was produced for multiple social media platforms.
